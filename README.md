## ‚ú® Key Features & Functionality (90% Complete)

SAMS is structured around a multi-tiered user environment, currently offering the following major functionalities:

### 1. Centralized Administrative Dashboard (Admin/Principal View)

* **Key Metrics:** Overview of Total Students and Total Staff counts.

* **Performance Monitoring:** Visualization of Grade-wise Average Results (e.g., Grade 6A, 6B, 6C, 7A).

* **Attendance Tracking:** Quick view of the last 5 days of school-wide attendance.

### 2. Core Administration Tools

* **User Management:** Tools for managing and assigning roles to staff and students.

* **Database Management:** Dedicated databases for Student and Teacher records.

* **School-Wide Reporting:** Generation of reports for Attendance and Academic Results.

### 3. Teacher Tools

* **My Classes:** Quick access to assigned classes.

* **Take Attendance:** Streamlined interface for recording daily student attendance.

* **Grade Assignments:** Functionality for submitting and managing grades/results.
